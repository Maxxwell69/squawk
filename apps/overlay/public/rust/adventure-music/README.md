# Rust — adventure music

Drop **MP3 / M4A / etc.** here with any filenames. Sorted **A→Z** is playlist order.

The **Rust adventure board** uses the same behavior as Sea of Thieves: **Start** on a mode that plays music picks a **random** first track, then plays through the rest until the list ends or you stop.

Served as `/rust/adventure-music/<filename>`.
